### Run server

1. 디렉토리 이동

    ```bash
    cd api-ai
    ```

2. 의존성 패키지 설치

    ```bash
    poetry install
    ```

3. 서버 실행

   ```bash
   poetry run uvicorn main:app --reload
   ```
